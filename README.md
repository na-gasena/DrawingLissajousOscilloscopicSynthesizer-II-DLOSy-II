# Drawing Lissajous Oscilloscopic Synthesizer II (DLOSy-II)

Web Audio APIを用いた、オシロスコープ描画（Lissajous / リサジュー図形）機能とステップシーケンサーなどを搭載した、ハードウェアライクな高機能Webシンセサイザーです。

---

## 概要

DLOSy-II（DLOSy20）は、ブラウザ上で動作する高機能な音楽制作・シンセサイザー環境です。視覚的かつ直感的な操作が可能なオシロスコープ（XY Scope）波形描画機能に加え、多彩なモジュレーションおよびシーケンス機能を搭載しています。

### 主な機能

- **高音質なシンセエンジン (`audio-engine.js`)**
  - 基本波形（Sine, Triangle, Square, Sawtooth）に加え、カスタム描画波形に対応。
  - カットオフ、レゾナンス、ボリューム等の主要パラメーターをリアルタイム制御。
- **直感的なADSRエンベロープ・エディター (`adsr-editor.js`)**
  - キャンバス上のドラッグ操作で直感的に調整可能なADSRエンベロープジェネレーター。
- **マルチトラック・シーケンサー & アルペジエーター (`step-sequencer.js`, `arpeggiator.js`)**
  - 16/32ステップのステップシーケンサー。
  - 各種スケール選択（Major, Minor, Pentatonic, Blues, Chromatic 等）およびトランスポーズ機能。
  - 自由度の高いアルペジエーター。
- **ドラムマシン (`drum-machine.js`)**
  - リズムセクションの構築が可能なステップ入力式ドラムシーケンサー。
- **VCOループ & パラメーターシーケンサー (`vco-loop.js`)**
  - 周波数、フィルター、エンベロープ等の変化をステップまたはパターンごとにループ制御・モジュレーションする機能。
- **波形描画 & XYスコープ機能 (`drawing-mode.js`)**
  - マウスやタッチ操作で描画したカスタム軌跡（波形）をXY Scope経由でオシロスコープ波形として再生・発音。
- **エフェクト・エンジン (`effects-engine.js`)**
  - ディレイ、フィードバック、リバーブ等の空間系・変調系エフェクト。
- **MIDI入出力およびCV同期 (`midi-in.js`, `midi-out.js`, `cv-clock.js`)**
  - Web MIDI APIによる外部MIDIキーボード・コントローラー入力、およびMIDIアウト対応。
  - アナログシンセ等との同期を可能にするCVクロック同期機能。
- **プリセット・マネージャー (`preset-manager.js`)**
  - 音色設定やシーケンスパターンのJSONエクスポート/インポート機能。

## 実行方法

本プロジェクトは静的ファイルとシンプルなNode.jsによるローカル開発サーバー（`http-server`）で構成されています。

### 前提条件

- [Node.js](https://nodejs.org/) (npm) がインストールされていること。

### セットアップと起動手順

1. **依存関係のインストール**
   ターミナル（PowerShellやCommand Promptなど）で本ディレクトリを開き、以下のコマンドを実行して必要なモジュールをインストールします。
   ```bash
   npm install
   ```

2. **ローカルサーバーの起動**
   以下のコマンドを実行し、ローカル開発サーバーを起動します。
   ```bash
   npm run dev
   # または
   npm start
   ```
   *※自動的にブラウザが開かない場合は、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスしてください。*

## 基本操作・演奏方法

1. **音声出力の有効化**
   ブラウザのセキュリティ制限のため、ページが開いた直後は音が出ません。**画面内の任意の場所をクリックまたはタップ**すると、オーディオエンジンが初期化され発音可能になります。

2. **キーボード演奏 (コンピューターキーボード対応)**
   キーボードのキーがピアノの鍵盤に対応しています。
   - **白鍵**: `S` (ド), `D` (レ), `F` (ミ), `G` (ファ), `H` (ソ), `J` (ラ), `K` (シ), `L` (ド)
   - **黒鍵**: `E` (ド#), `R` (レ#), `T` (ファ#), `Y` (ソ#), `U` (ラ#), `I` (ド#)

3. **つまみ（Knob）の操作**
   画面上のつまみをクリックまたはドラッグ（上下方向）することで、パラメーター値を細かく調整できます。

4. **画面・タブの切り替え**
   - センターパネル上のタブ（`SEQUENCER`, `DRUMS`, `ARP`, `GLYPH`）をクリックして各機能にアクセスできます。
   - キーボードの `Tab` キーを押すことで、フォーカスされているパネルのアクティブなタブやスロット、パラメータ表示を素早く切り替えることができます。
   - `VCO LOOP` または `DRAWING` パネルがアクティブな場合、キーボードの `1` 〜 `8` キーでパターン/スロットを瞬時に切り替えられます。
